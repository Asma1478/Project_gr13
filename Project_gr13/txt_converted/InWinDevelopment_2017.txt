         ∣董事長的話
         迎廣科技股份有限公司投入 pc 機殼製造及銷售的領域已有 30 多年的時間，本公司持續在機殼領域中創新研發，近年來每一個年度都有不同的形象機殼推陳出新、突破機
目錄       殼設計舊框架的創意產品，己經塑造了機殼產業中創新突破領先廠商地位，更得到德國 if 設計大奬、台灣精品奬等無數的獲獎認同。本公司的主要產品包括 pc 機殼、ipc/
         server 機殼、電競機殼、電源供應器、電競電源供應器……等產品，2016 年度營業收入達 2,056,224 千元新台幣。
關於本報告書
         對迎廣科技而言，企業責任不僅是為了股東利益而存在，而是要將環境、社區、顧客、員工及供應鏈等所有利害關係人融入日常經營的考量中。「工業發展與環境保護並
策略與願景    行不悖」是我們多年來堅持的理念，制訂環保與企業社會責任政策，全面落實推行環安衛管理與企業社會責任政策，持續改善是我們推動企業經營的方針。本公司已通過
         iso 14001 環境管理系統驗證、qc080000 有害物質流程管理系統驗證，除此之外，2015 年本公司更推動 iso50000 能源管理系統，並於當年度取得 sgs 的 iso50001
公司簡介     能源管理系統的驗證通過，不僅如此，本公司也從產品的開發階段即配合客戶推動綠色管理計劃，進行綠色產品開發，推動綠色產品及回收機制，並符合 weee、rohs、
         reach、energy star、epeat 等國際環保法規的要求。
環境永續發展
         本公司本於「取諸於社會，用諸於社會」的企業精神，長期捐贈桃園市公埔國民小學、大溪僑愛國民小學經費，成立管樂社團，培養小學生的音樂素養，進而陶冶學生的身、
         心、靈健全發展，另外，本公司也相當重視社區和諧，不時贊助本公司所在地村、里中心的節慶活動經費，種種措施都顯示本公司在企業社會責任管理工作的推動方面，全
企業社會責任
         力以赴，矢志不移，所以，我們所獲得的績效成果，也非常豐碩，十分亮麗。截至本報告書完成之日為止，本公司從未發生因為產生之活動、產品及服務，發生重大污染狀況。

附錄       根據國際研究暨顧問機構 gartner 資料，2016 年 pc 市場持續低迷，由於技術升級的吸引力不足以帶動實質市場成長，pc 市場全面呈現停滯狀態。再者，因消費者使用型
         態轉變，大幅提升智慧行動裝置的使用率，相對排擠傳電腦的購買意願，這表示著 pc 產業從 2012 年的出貨高峰以來，依舊持續呈現衰退的走勢，本公司經營階層將調整
         策略及營運方向，因應電競市場及 iot 物聯網的趨勢發展，全力衝刺 ipc/server 機殼、電競機殼等相關的產品線。對本公司而言，產業、景氣的變動是必然，而我們必須
         永遠先做好準備，目前公司經營方針是繼續強化品牌營運，擴大零售市場之佔有率、重整雲端產品供應鏈，強化軟硬體整合能力，提昇產品競爭力及擴大市場、繼續擴大自
         動化生產及電子化管理之能力、提昇雲端系統之研發及驗證能力，以提昇公司在此產業的市場地位、強力擴大大陸內需市場之營運規模及品牌地位、強化 openstack 軟體
         設計能力，加強市場動能，以提昇營業實績，並善盡公民的企業社會責任。自 1985 年當責迎廣營運以來，我本人和經營團隊念茲在茲的就是重視與維護員工權益，要竭盡
         所能，全力照顧好同仁。

         落實環保、安全衛生與勞動平權工作以及透明的進行資訊揭露，是企業責任所在。 因應未來業務成長的需要，迎廣科技將戮力強化組織，投入人才培育，優化關鍵流程，致
         力創新技術研發，並透過資訊及資源的有效整合，持續追求高品質、高效率、高環保、低成本，創新求變，成為機殼產業永遠的領導廠商。展望未來，我們經營團隊不僅將
         信守環保與企業社會責任政策中所揭示的企業公民宣言以及各項承諾，奮力向前，在此同時，迎廣也會繼續秉持珍惜自然資源的精神，朝污染預防及持續改善的方向不斷努
         力；有效控制安全衛生風險，以零災害為目標，努力不懈；堅持「清潔生產、綠色製程、維護人權」的作法，展現善盡企業社會責任的決心；朝模範企業社會公民的遠景，
         大步邁進。

         ∣公司治理
         有鑒於國際上績優企業瀆職及破產案件仍層出不窮，再加上金融危機後全球經濟尚未走上完全復甦之路，相對地，全球消費者、投資者與企業夥伴對於良好企業治理與永續
目錄       企業實踐的期望與日俱增。而企業成功經營的關鍵，除了經營者與員工的全心投入之外，尚必須仰賴公司股東的支持；財務機制透明化與股東權益維護提升，更是迎廣贏得

         ∣環境管理方針揭露
         ( 一 ) 環境面管理方針揭露 ( 環境績效管理方針之揭露 )
目錄       迎廣建置 iso14001 管理系統，希望藉由產品面、活動面、服務面執行有效的永續發展策略，做為經營的方針與目標。提供給客戶的各項產品，其設計與製造過程亦均符合
         環保要求，由環境管理、增資環境績效，進而改善地球的汙染、減少環境衝擊、普及環安意識、節約能源，有效利用資源…等等，最終達成正面的環境經營的理念與推展執
關於本報告書   行的績效，履行企業社會責任，達成永續經營的目標。

策略與願景    ( 二 ) 環境宣言及環境政策
         為追求人類生命之延續及永保地球綠色環境，本公司除積極落實企業環保責任與使命外，在產品設計、使用與廢棄等活動中致力於降低對整體環安衛之衝擊，並致力於污染
公司簡介     防治與改善，並應符合相關法規規定，透過全員參與及承諾，來達成環境保護與企業之永續發展。本公司於 iso14001 管理系統中，揭露環境宣言及環境政策如下：

         1. 減少環境衝擊 ：
環境永續發展
           採用適當的生產或污染防制之技術，並增加產品的節能減廢設計，以預防產品使用後產生污染，更有效的利用能資源或減少在產品活動及服務過程中所引起的環境衝擊。
         2. 善盡環保責任 ：
企業社會責任
           為滿足客戶綠色產品之要求，對於原 ( 物 ) 料、零件、輔料或包裝資材，廢除和消減相關之環境管制物質，採取禁用或限用之原則，保護地球環境以及消減對生態系統之
           影響。
附錄
         3. 力行資源回收 ：
           力行能資源回收活動與教育訓練，以提昇愛惜能資源與環安衛意識。
         4. 推動工業減廢 ：
           導入環保設計理念，以便有效的利用能資源或減少在產品活動及服務過程中所引起的環境衝擊，以符合客戶之產品環保要求。
         5. 遵守環保法規 ：
           遵守環保法令規章及其他要求事項，符合政府環保法令，善盡企業社會責任因應國際環保趨勢，致力於綠色產品研發，為滿足客戶對綠色環保產品之要求。
         6. 落實環境管理 ：
           普及環保教育，全員參與環境管理系統之推行，藉著持續性的稽核活動及環境管理審查，發現缺點，持續改善，以建立合法、有效的環境管理制度，以利環境保護活動之
           推行。
         7. 普及環安意識：
           以追求幸福為企業經營的目標，為維護健康生活環境及員工能安心工作之安全環境，採用適當的風險控制技術，以有效降低職災風險及損失，藉由危害預防、持續改善，
           制定目標及方案，持續改善環境保護與職業安全衛生績效；實施環安衛教育及宣導，提昇員工環安衛意識，促進參與並落實環保安全衛生工作；符合政府環保及職業安全
           衛生法令，構築一個安全無慮工作環境的同時，致力追求零污染、零傷害、零職業病、零火災事故，並持續改善，以提昇工作安全及環境品質。

公司簡介     ( 一 ) 勞工 ( 勞工實踐與合理工作面管理方針之揭露 )
         迎廣認為員工是企業最重要的資產，對此，本公司致力打造一個完善的職場勞動環境。除了提供同仁具競爭力的薪資、全方位的教育訓練制度、多樣的員工福利以及安全衛生的工
環境永續發展   作環境外，也提供所有員工在食、衣、住、行、育、樂等各方面完善優質的生活環境，期許每一位同仁樂在工作、無後顧之憂，對此，本公司管理作為如下：

         1. 安全、衛生的工作環境
企業社會責任
         迎廣致力以零災害為目標，符合工安法令要求，並以前瞻性眼光積極提昇各安全層面；更持續對員工進行安衛教育及宣導，鼓勵主動參與安衛活動，潛移默化全體同仁安衛理念與
附錄       習慣。本公司對於安全衛生的政策如下：
          符合工安法令要求，以零災害為目標，致力公司在安全第一下穩健發展。
          運用先進安衛、風險及救災技術，以前瞻性眼光積極提昇各安全層面。
          落實預防管理制度，確保工作環境及作業安全。
          明確規範各級人員的安衛責任，以共同消除危害因子，預防事故發生。
          持續進行安衛教育及宣導，潛移默化全員安衛理念與習慣。

         2. 員工健康與活動
         迎廣長期致力於營造身、心、靈健康職場環境，除協助員工進行全方位的健康管理及健康促進外，更進而將關懷的觸角延伸至員工家庭，希望藉此達到提昇工作士氣、提高工作品
         質的目的。

         3. 人才培訓與教育訓練
         全方位的人才發展與培育是迎廣提昇人力素質的關鍵。迎廣的人才培育政策，乃依員工之執掌、職級量身規劃每位員工於職涯發展中所需之訓練課程，包括新進人員、專業能力、
         管理才能訓練、語文培訓等課程，並搭配豐富的 ojt 及在職進修等學習資源，以提供員工一個能持續學習成長的工作環境。

策略與願景
         為提升同仁安全衛生意識並能因應各種突發而來緊急事故，公司已建立一套完整的緊急應變機制，包括通報機制、緊急應變編組、應變器材、教育訓練等，以強化公司同仁的應
公司簡介     變能力。

         迎廣在 ohsas18001 框架下，遵循勞安通報系統，定期通報勞檢單位的失能傷害頻率及失能傷害嚴重率統計百分比。據統計，2016 年迎廣公司職業災害 ( 不含交通意外 ) 統計
環境永續發展   結果，台灣廠區失能傷害頻率為 7.06，失能傷害嚴重率為 57，綜合災害指數為 0.63。

企業社會責任   迎廣在作業安全衛生管理面向，除了有高風險作業管制、承攬商入廠管理及施工安全管制、化學品安全管理及安全稽核制度等外，也妥善規劃災害緊急應變程序及定時演練，務
         使萬一災害發生時，對公司財物、人員、社會、環境的損傷與衝擊降至最低。

附錄       迎廣持續透過建立安全文化來提供安全舒適的工作環境。所有職業災害個案均進行原因分析，研擬與執行改善方案，並且定期統計分析職災發生率較高的單位與職災類型，其為
         嚴重度相對較高或屬跨單位、重複發生性質者列為教育訓練與管理重點。

         除工安議題外，迎廣對於員工之健康議題更為關注，要使公司能茁壯成長首要工作就是照顧、關心員工健康，辦理健康檢查，另為使產後女性員工能安心工作，亦增設集（哺）

         ∣社區、社會關懷與回饋
         迎廣了解企業的成功除了自身的努力外，仍需要一個祥和社會的支持，才有可能使公司業績飛躍成長與展現蓬勃的活力。因此，迎廣本身暸解在致力於公司營業績效外，亦要善
目錄       盡一己之力，對社會有所回饋。迎廣對於社會的回饋是以「教育」、「藝文」、「體育」、「公益」、及「環保」為主軸，有系統的從事社會公益。
         早在本公司成立之初就已經啟動社會關懷活動，我們也期待透過企業全員參與的形式回饋社會、貢獻己力。
關於本報告書
         即使 2008 年以來的全球景氣波動，對於公司的營收多少造成衝擊，但就迎廣科技而言，不論營業狀況之好壞，都不忘記回饋鄉里，回饋社會。迎廣高階主管理解到：公司能穩
策略與願景    健成長，主要來自全體同仁的努力與社會大眾的支持，因此公司除獎勵員工的努力之外，身為社會公器的企業，也應善盡企業的社會責任，回饋社會大眾，對此，迎廣秉持著「取
         之於社會，用之於社會」的理念，對教育及弱勢團體的援助向來不遺餘力，2016 年社會公益與捐助金額逾 137 萬元。
